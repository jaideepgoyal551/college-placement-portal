🎓 College Placement Portal

A full-stack College Placement Portal that streamlines the campus recruitment process by connecting Students, Recruiters, and Placement Officers on a single platform. The portal simplifies job postings, applications, interview scheduling, and placement management with a modern, responsive interface.

---

✨ Features

👨‍🎓 Student

- Secure authentication and profile management
- Upload and manage resumes
- Browse and apply for jobs & internships
- Track application status
- View interview schedules
- Receive placement notifications
- Personalized dashboard

🏢 Recruiter

- Company profile management
- Post job and internship opportunities
- Define eligibility criteria
- View eligible applicants
- Download student resumes
- Shortlist or reject candidates
- Schedule interview rounds
- Publish recruitment results

👨‍💼 Placement Officer

- Manage students and recruiters
- Verify company registrations
- Publish placement notices
- Monitor applications
- View placement statistics
- Generate reports
- Manage interview schedules

⚙️ Installation

Clone the repository

git clone https://github.com/your-username/placement-portal.git
cd placement-portal

Backend Setup

cd server
npm install

Create a ".env" file:

DATABASE_URL=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

Run migrations:

npx prisma migrate dev

Start the backend:

npm run dev

Frontend Setup

cd client
npm install
npm run dev
